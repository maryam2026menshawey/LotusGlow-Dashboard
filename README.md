# Project Overview

This project is an interactive Business Intelligence dashboard developed in **Power BI** to analyze sales performance, customer purchasing behavior, category trends, and brand market share for LotusGlow.

The dashboard transforms raw transactional data into meaningful business insights using KPI tracking, dynamic filtering, and interactive visualizations.

This project demonstrates practical skills in:
- Data Cleaning & Transformation
- Data Modeling
- DAX Calculations
- KPI Development
- Business Intelligence Reporting
- Data Visualization
- Dashboard Design

---

# Business Problem

Businesses often struggle to:
- Monitor sales performance efficiently
- Identify top-performing customers and brands
- Track monthly sales trends
- Compare category performance
- Understand customer purchasing behavior

This dashboard helps stakeholders make data-driven decisions through interactive analytics and visual storytelling.

- Payment methods vary across product categories.
